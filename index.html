A single input/output parcel distributor

ASD machine is a revolutionary parcel dispenser. The purpose of this machine is to provide e-retailers a new way to tackle the "last mile" problem. The distributor is totally automated from the introduction of the parcel to the restitution of it. Moreover, the system was thought in order to optimize internal storage space needed and to minimize commercial visible space of the machine input/output.

Automated Secured Delivery Machine

Parcel Storage

The machine stores different sizes packages of maximum 25kg and 50cm x 50cm x 70cm .

Modular Distributor Conception

The machine can be installed into rooms with different dimensions without any trouble.

100% User Friendly

User interface such as mobile application are developped under Android to increase customer experience.

Ultra Secured Installation

Neither customers nor postmen can access the storage room.
